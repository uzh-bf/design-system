{/* START README */}
The `Resizable` component provides user-controlled resizable panels with drag handles, enabling flexible layout customization. Users can adjust panel sizes by dragging resize handles to create personalized workspace arrangements.

Use this component when you need to:
- Create split-pane layouts that users can customize
- Build resizable dashboard panels or content areas
- Implement adjustable sidebars or content sections
- Provide flexible workspace layouts that adapt to user preferences

The resizable system supports both horizontal and vertical layouts, nested panel groups, minimum/maximum size constraints, and smooth resize animations. It maintains responsive behavior and provides accessible keyboard navigation for resize operations.
