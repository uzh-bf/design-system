{/* START README */}
The `ToggleGroup` component provides a group of toggle buttons that work together, allowing users to select one or multiple options from a set. It's perfect for formatting controls, view options, or any scenario where users need to toggle between related choices.

Use this component when you need to:

- Create formatting toolbars with options like bold, italic, underline
- Provide view switching controls (list, grid, card views)
- Enable multiple selection filters or options
- Group related toggle controls with consistent behavior

The toggle group supports both single and multiple selection modes, maintains proper keyboard navigation, and provides clear visual feedback for active states. It automatically manages the state relationships between toggle items.
