{/* START README */}
The `Tag` component provides a simple, consistent way to display labels, categories, or status indicators. It offers clean styling with customization options and maintains accessibility standards throughout your interface.

Use this component when you need to:

- Display categories, topics, or classification labels
- Show status indicators or metadata
- Create content tags for articles or posts
- Label items in lists or cards
- Build filter or selection interfaces

The component provides consistent typography, proper spacing, customizable styling through className overrides, and standard testing attributes for reliable automation.

<br />

The Tag component accepts the following props:

- @param id - The id of the tag.
- @param data - The object of data attributes that can be used for testing (e.g. data-test or data-cy)
- @param label - The label of the tag.
- @param className - The optional className object allows you to override the default styling.
