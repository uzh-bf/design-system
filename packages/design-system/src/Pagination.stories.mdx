{/* START README */}
The `Pagination` component provides navigation controls for multi-page content, allowing users to move between different pages of data or content. It includes previous/next buttons, page numbers, and ellipsis indicators for large page ranges.

Use this component when you need to:

- Navigate through large datasets split across multiple pages
- Provide clear indicators of current page position and total pages
- Enable users to jump to specific pages or navigate sequentially
- Display page controls in tables, search results, or content listings

The pagination component automatically handles active states, provides accessible navigation, and includes ellipsis indicators when there are many pages. It supports both controlled and uncontrolled usage patterns.
