{/* START README */}
The `Sheet` component provides slide-out panels that appear from screen edges for secondary content, forms, or navigation. It offers an alternative to modals when you need to maintain context with the main content.

Use this component when you need to:
- Create mobile-friendly navigation menus
- Display detailed forms or settings panels
- Show additional information without full page navigation
- Provide contextual actions or filters for the main content

The sheet slides in from configurable directions (right, left, top, bottom), includes proper focus management, and provides structured sections for headers, content, and footers.
{/* END README */}
