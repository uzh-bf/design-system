{/* START README */}
The `Badge` component displays small status indicators, labels, or notification counts with consistent styling and visual hierarchy. It provides a compact way to highlight important information or categorize content.

Use this component when you need to:

- Show status indicators (verified, premium, new, etc.)
- Display notification counts or numbers
- Label categories, tags, or classifications
- Highlight important attributes or properties

The badge supports multiple variants (default, secondary, destructive, outline) and can include icons or be styled as circular counters. It's designed to be visually distinct while remaining unobtrusive.
