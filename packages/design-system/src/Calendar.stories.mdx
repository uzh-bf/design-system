{/* START README */}
The `Calendar` component provides an interactive date picker interface with month/year navigation and date selection capabilities. It offers a familiar calendar grid layout for users to browse and select dates.

Use this component when you need to:
- Allow users to select single or multiple dates
- Provide date browsing and navigation interfaces
- Create date range selection functionality
- Display date-related information in calendar format

The calendar supports different selection modes (single, multiple, range), disabled dates, and custom styling. It includes keyboard navigation and accessibility features for screen readers.
