{/* START README */}
The `ShadcnLabel` component provides accessible form labels with proper semantic associations to form controls. It ensures that form inputs are properly labeled for both visual users and screen readers.

Use this component when you need to:

- Label form inputs, checkboxes, and radio buttons
- Create accessible forms with proper semantic associations
- Provide clear, consistent labeling across form elements
- Ensure proper focus management and screen reader support

The label component automatically associates with form controls through the `htmlFor` attribute, provides consistent typography, and maintains proper accessibility standards throughout your forms.

**Note**: This is prefixed as "ShadcnLabel" to distinguish it from the standalone Label component in the design system.
{/* END README */}
