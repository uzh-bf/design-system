{/* START README */}
The `Breadcrumb` component provides hierarchical navigation that shows users their current location within a website or application structure. It helps users understand where they are and provides an easy way to navigate back to parent pages.

Use this component when you need to:
- Show navigation hierarchy in multi-level applications
- Help users understand their current location context
- Provide quick navigation back to parent sections
- Improve wayfinding in complex information architectures

The breadcrumb supports collapsible sections with ellipsis for long paths, clickable links for navigation, and proper accessibility with ARIA labels and semantic markup.
{/* END README */}
