The `Prose` component provides automatic typography styling for content-heavy areas like articles, documentation, or rich text. It applies consistent formatting to HTML elements without requiring manual styling of each element.

Use this component when you need to:

- Style content from markdown or CMS systems
- Display articles, blog posts, or documentation
- Format user-generated content consistently
- Create readable typography without element-by-element styling
- Ensure consistent spacing and hierarchy in text content

The component automatically applies appropriate typography styles to headings, paragraphs, links, lists, and other common content elements, maintaining design system consistency throughout your text content.

<br />

The Prose component accepts the following props:

- @param id - The id of the prose component.
- @param data - The object of data attributes that can be used for testing (e.g. data-test or data-cy)
- @param children - The content of the prose component.
- @param className - The optional className object allows you to override the default styling.
