The `ShadcnDropdownMenu` component provides comprehensive dropdown menus with nested submenus, keyboard shortcuts, and organized content sections. It creates feature-rich context menus and action panels triggered by buttons or other elements.

Use this component when you need to:

- Create user account menus with profile actions and settings
- Build context menus with multiple action groups
- Implement nested menu structures with submenus
- Provide organized action lists with keyboard shortcuts

The dropdown menu supports nested submenus, separator groupings, disabled states, keyboard navigation, and customizable positioning. It automatically handles focus management and provides proper accessibility support.

**Note**: This is prefixed as "ShadcnDropdownMenu" to distinguish it from the custom Dropdown component in the design system.
