{/* START README */}
The `AspectRatio` component maintains consistent width-to-height ratios for media content and containers. It ensures responsive design by automatically scaling content while preserving the specified aspect ratio across different screen sizes.

Use this component when you need to:

- Display images or videos with consistent proportions
- Create responsive media containers that scale properly
- Maintain layout consistency for cards or thumbnails
- Prevent content layout shifts during loading

The component accepts a ratio prop (e.g., 16/9, 4/3, 1/1) and wraps any content to maintain that proportion. It's particularly useful for images, videos, and placeholder content.
