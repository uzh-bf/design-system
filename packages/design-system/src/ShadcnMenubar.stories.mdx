{/* START README */}
The `ShadcnMenubar` component provides a horizontal menu bar with dropdown menus, similar to application menu bars found in desktop software. It supports nested menus, keyboard shortcuts, checkboxes, radio groups, and organized menu sections.

Use this component when you need to:
- Create application-style menu bars with multiple top-level menus
- Implement complex menu structures with nested submenus
- Provide keyboard shortcuts and accelerators for menu actions
- Build desktop-like interfaces with traditional menu navigation

The menubar supports full keyboard navigation, maintains focus management, includes checkbox and radio group functionality, and provides consistent styling for complex menu hierarchies.

**Note**: This is prefixed as "ShadcnMenubar" to distinguish it from other navigation components in the design system.
{/* END README */}
