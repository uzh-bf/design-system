{/* START README */}
The `Carousel` component provides a touch-friendly, responsive slideshow interface for displaying multiple items in a horizontal scrolling layout. It enables users to browse through content collections with smooth navigation controls.

Use this component when you need to:

- Display product galleries or image collections
- Show featured content or testimonials in rotation
- Create space-efficient browsing for multiple items
- Present step-by-step tutorials or onboarding flows

The carousel supports touch/swipe gestures, keyboard navigation, and includes previous/next buttons. It automatically handles responsive behavior and can display multiple items simultaneously.
{/* END README */}

export const Default = () => {
  return (
    <Carousel className="w-full max-w-xs">
      <CarouselContent>
        {Array.from({ length: 5 }).map((_, index) => (
          <CarouselItem key={index}>
            <div className="p-1">
              <Card>
                <CardContent className="flex aspect-square items-center justify-center p-6">
                  <span className="text-4xl font-semibold">{index + 1}</span>
