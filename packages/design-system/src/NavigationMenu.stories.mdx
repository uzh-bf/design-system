The `NavigationMenu` component provides sophisticated horizontal navigation with dropdown panels for complex site hierarchies. It combines main navigation items with rich content panels that can display detailed information, links, and organized sections.

Use this component when you need to:

- Create comprehensive site navigation with multiple levels
- Display rich content previews or detailed information in dropdown panels
- Organize complex navigation structures with categories and subcategories
- Provide both simple links and interactive content areas in navigation

The navigation menu supports keyboard navigation, automatic positioning, smooth animations, and flexible content layouts. It's ideal for header navigation in applications with extensive feature sets or content hierarchies.
