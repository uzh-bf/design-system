{/* START README */}
The `Avatar` component displays user profile pictures with automatic fallback to initials when images fail to load. It provides a consistent way to represent users throughout the application with proper accessibility and loading states.

Use this component when you need to:

- Display user profile pictures in headers, comments, or lists
- Show team member avatars in collaboration interfaces
- Create user identification in chat or messaging systems
- Represent entities with visual identity (users, organizations, etc.)

The avatar automatically handles image loading failures by displaying fallback initials. It supports different shapes (circular, rounded) and can be grouped together for team displays.
