{/* START README */}
The `ContextMenu` component provides right-click context menus with rich interactive options including submenus, checkboxes, radio groups, and keyboard shortcuts. It enables contextual actions directly related to specific UI elements.

Use this component when you need to:

- Provide contextual actions for specific elements or regions
- Create right-click menus for tables, images, or content areas
- Offer quick access to relevant commands based on context
- Implement desktop-like interaction patterns in web applications

The context menu supports nested submenus, different item types (actions, checkboxes, radio groups), keyboard shortcuts, and proper accessibility with ARIA attributes.
{/* END README */}
