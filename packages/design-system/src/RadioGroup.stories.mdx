{/* START README */}
The `RadioGroup` component provides mutually exclusive selection options where users can choose only one option from a set. It ensures proper form behavior and accessibility for single-choice scenarios.

Use this component when you need to:
- Present mutually exclusive options (settings, preferences, choices)
- Create single-selection forms or configuration interfaces
- Provide clear visual indication of current selection
- Ensure only one option can be selected at a time

The radio group manages selection state automatically and provides proper ARIA labeling and keyboard navigation. Each radio item should be paired with a descriptive label.
{/* END README */}
