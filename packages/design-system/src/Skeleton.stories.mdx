import { Skeleton } from './Skeleton'

{/* START README */}
The `Skeleton` component provides placeholder loading states that mimic the shape and size of content while data is loading. It creates a better perceived performance by showing content structure before actual content loads.

Use this component when you need to:
- Show loading states for content that takes time to load
- Improve perceived performance during data fetching
- Maintain layout stability while content loads
- Create smooth transitions from loading to loaded states

The skeleton automatically animates with a subtle shimmer effect and can be styled to match any content shape (text lines, circles, rectangles, etc.).
{/* END README */}
