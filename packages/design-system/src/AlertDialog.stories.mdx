{/* START README */}
The `AlertDialog` component provides modal confirmation dialogs for critical user actions that require explicit confirmation. It ensures users are aware of potentially destructive or irreversible actions before proceeding.

Use this component when you need to:

- Confirm destructive actions (delete, remove, clear data)
- Get explicit user consent for irreversible operations
- Display critical warnings that require user acknowledgment
- Interrupt user flow for important decision points

The dialog includes structured sections for title, description, and action buttons (cancel/confirm). It provides keyboard navigation, focus management, and proper accessibility with ARIA attributes for screen readers.
{/* END README */}
