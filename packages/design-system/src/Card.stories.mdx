{/* START README */}
The `Card` component provides a flexible container for grouping related content with consistent spacing, borders, and visual hierarchy. It serves as a foundational layout component for organizing information into digestible sections.

Use this component when you need to:

- Group related content into visually distinct sections
- Create product cards, profile cards, or information panels
- Organize dashboard widgets or feature highlights
- Structure form sections or settings panels

The card includes structured sections for headers, content, and footers with proper spacing and typography. It provides a clean, bordered container that works well in grids or lists.
{/* END README */}
