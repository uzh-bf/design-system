{/* START README */}
The `Separator` component provides visual dividers to separate content sections both horizontally and vertically. It creates clear visual boundaries between different areas of content while maintaining proper semantic structure.

Use this component when you need to:

- Visually separate different sections of content
- Create dividers in navigation menus or toolbars
- Add structure to lists or grouped information
- Provide visual breaks between content areas

The separator supports both horizontal and vertical orientations, integrates seamlessly with screen readers for accessibility, and maintains consistent styling across different contexts. It's subtle enough to organize content without being visually distracting.
