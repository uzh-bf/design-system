{/* START README */}
The `Command` component provides a command palette interface for quick navigation and action execution. It enables users to search and execute commands through a keyboard-driven interface, similar to VS Code's command palette.

Use this component when you need to:

- Create application-wide command and navigation interfaces
- Provide quick access to features and actions via search
- Build keyboard-driven workflows for power users
- Implement global search functionality with actions

The command palette supports grouped commands, keyboard shortcuts, search filtering, and empty states. It includes proper ARIA labeling and keyboard navigation for accessibility.
{/* END README */}
